本章数据缺口密度高：

- **数据缺口 G42 / HUMAIN / 印度算力规模**——大量项目仍处于公告 / 执行早期，按"已公告 + 待落地 + 业内估算"分层

- **G42 与 Mubadala 资金链条非完全公开**——按公开披露
- **HUMAIN 与 PIF 资金链条非完全公开**——按公开披露
- **印度 GPU 进口规模**——海关数据不完全公开，业内估算
- **马来西亚 Johor DC 实际客户结构**——大多 colocation 客户保密，按业内估算 + 区间
